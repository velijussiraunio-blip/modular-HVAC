# Modular Service Trunk – Parametric Design Tool

This repository contains a simple Python-based engineering tool to generate a parametric layout drawing of a 6 m modular MEP service trunk (QTS-style).

## Files
- service_trunk.py – main design script
- requirements.txt – Python dependencies

## Requirements
- Python 3.9+
- matplotlib

## How to Run
```bash
pip install -r requirements.txt
python service_trunk.py
```

## Purpose
Internal engineering concept design and standardization tool.
Not a production or fabrication drawing.
